Red FeedForward para problema XOR¶

El problema XOR no puede ser solucionado por un perceptrn simple. Para esto, se introducen capas ocultas que ayudan a transformar los datos de entrada y que se puede determinar regiones de decisin adecuadas. Se ha probado que una red neuronal con una capa oculta es un aproximador universal, lo que quiere decir que, tericamente, es capaz de aproximar cualquier funcin de manera adecuada.

Solución del problema XOR por funciones lógicas básicas¶

El problema XOR puede expresarse por medio composicin de funciones lgicas bsicas o perceptrones, como:

$$XOR(x_1,x_2) = AND[NOT(AND(x_1,x_2)), OR(x_1,x_2)]$$

As, podemos definir varios perceptrones para solucionar este problema.

In [3]:
#Perceptrn AND
def AND(x):
    w = np.array([1,1])
    return 1*(np.dot(x,w)-1.5 > 0)

#Perceptrn OR
def OR(x):
    w = np.array([1,1])
    return 1*(np.dot(x,w)-0.5 > 0)

#Perceptrn NOT x_1
def NOT(x):
    try:
        w = np.array([-1,0])
        return 1*(np.dot(x,w) + 0.5 > 0)
    except:
        return 1*(-x + 0.5 > 0)

Estas funciones, entonces, clasifican los datos segn cada uno de los problemas.

Entrenamiento de la red¶

Definiremos una red con una sala capa oculta. Basta con que esta capa oculta tenga tres dimensiones. Esta capa oculta, adems tendra una activacin por medio de la tangente hiperblica. Entonces, la podemos definir como:

$$h= \tanh(W^{(1)}\cdot x + b^{(1)})$$

La capa de salida estar dada por la funcin softmax. En este caso, determinamos la funcin Sofmax como:

$$f_i(x) = \frac{\exp(o_i-\max(o))}{\sum_k \exp(o_k-\max(o))}$$

En este caso, tenemos dos neuronas de salidas, pues tenemos dos clases (0 y 1). La clase que se eligir ser aquella que maximice la probabilidad.

Para actualizar los pesos, utilizaremos el algoritmo de backpropagation y el gradiente descendiente. Dado que la funcin Softmax determina probabilidades de clase, utilizaremos la entropa cruzada como funcin de riesgo:

$$R(\theta) = \sum_x y_x \log f(x)$$
In [7]:
np.random.seed(0)

#HIPERPARMETROS
#Nmero de dimensiones
h_dim = 3
#Rango de aprendizaje
l = 1.0
#Regularizacin
reg = 0.01
#Nmero mximo de iteraciones
max_it = 1000

#Inicializacin de parmetros de la red
w1 = np.random.rand(n,h_dim)/np.sqrt(n)
b1 = np.ones(h_dim)
w2 = np.random.rand(h_dim,2)/np.sqrt(h_dim)
b2 = np.ones(2)

t=1 #Primera iteracin
stop = False #Condicin de paro
while  stop == False:
    #FORWARD
    #Pre-activacin capa oculta
    a = np.dot(X,w1) + b1
    #Activacin capa oculta
    h = np.tanh(a)
    #Pre-activacin capa de salida
    out = np.dot(h,w2)+b2
    #Funcin softmax
    exp = np.exp( out ) #- np.max(out)) #Resta del mximo
    f = exp/exp.sum(1, keepdims=True)
    
    #BACKPROP
    #Variable de capa de salida
    d_out = f.copy() #Derivada softmax(j)-1  0 (dependiendo de la clase)  exp/exp.sum(1, keepdims=True)
    d_out[range(m),Y] -= 1 #Para funcin Softmax de salida funcin de riesgo entropa cruzada
    #Variable de capa oculta
    d_h = (1-h**2)*np.dot(d_out,w2.T)
    
    #Derivada sobre w2
    dw2 = np.dot(h.T,d_out)
    #Derivada sobre b2
    db2 = d_out.sum(0)
    #Derivada sobre w1    
    dw1 = np.dot(X.T,d_h)
    #Derivada sobre b1
    db1 = d_h.sum(0)
    #Derivada regularizacin
    dw2 += reg*w2
    dw1 += reg*w1
    
    #Actualizacin de pesos
    w2 -= l*dw2
    b2 -= l*db2
    w1 -= l*dw1
    b1 -= l*db1
    
    #Clculo de errores cometidos
    error = ((np.argmax(f, axis=1)-Y)**2).sum(0)
    if t%10 == 0: 
        print('poca', t, '\t Error', error, np.argmax(f, axis=1), '\t Entropy', -np.log(f[range(m),Y]).sum(0))
    
    #Aumenta la iteracin
    t += 1
    #Condicin de paro
    if error == 0 or t == max_it:
        stop = True
        print('poca', t, '\t Error', error, np.argmax(f, axis=1), '\t Entropy', -np.log(f[range(m),Y]).sum(0))

Visualización¶

La capa oculta puede verse como una transformacin sobre los datos originales. En el caso del problema XOR, los datos en dos dimensiones han pasado a un espacio de tres dimensiones, donde se pueden trazar regiones de decisin de las clases.

plt.show()

Entonces, una red FeedForward nos permite resolver el problema XOR a partir de aplicar una transformacin a los datos de entrada. Las redes FeedForward son una herramienta poderosa, que, sin embargo, requiere de una eleccin adecuada de hiperparmetros, adems de mayor capacidad de cmputo que otro tipo de algoritmos.

Agregar capas ocultas¶

Si bien una red FeedForward con una sola capa es suficiente, en el apartado anterior hemos utilizado una tcnica no supervisada de reduccin de dimensionalidad. Ahora, agregaremos una capa ms para que los datos en 3 dimensiones sean transformados a un espacio 2-dimensional (lo que facilita la visualizacin de la decisin de la red). En este sentido, construiremos una red compuesta de dos funciones o transformaciones:

$$X \to \mathbb{R}^3 \to \mathbb{R}^2 \to Y$$

np.random.seed(0)

#HIPERPARMETROS
#Nmero de dimensiones
h_dim = 3
o_dim = 2
#Rango de aprendizaje
l = 0.05
#Regularizacin
reg = 0.0
#Nmero mximo de iteraciones
max_it = 1000

#Inicializacin de parmetros de la red
w1 = np.random.rand(n,h_dim)/np.sqrt(n)
b1 = np.ones(h_dim)
w2 = np.random.rand(h_dim,o_dim)/np.sqrt(h_dim)
b2 = np.ones(o_dim)
w = np.random.rand(o_dim,2)/np.sqrt(o_dim)
b = np.ones(2)

t=0 #Primera iteracin
stop = False #Condicin de paro
while  stop == False:
    #FORWARD
    #Pre-activacin capa oculta
    a = np.dot(X,w1) + b1
    #Activacin capa oculta 1
    h1 = np.tanh(a)
    #Activacion capa oculta 2
    h2 = np.dot(h1,w2) + b2
    #Pre-activacin capa de salida
    out = np.dot(h2,w)+b
    #Funcin softmax
    exp = np.exp( out ) #- np.max(out)) #Resta del mximo
    f = exp/exp.sum(1, keepdims=True)
    
    #BACKPROP
    #Variable de capa de salida
    d_out = exp/exp.sum(1, keepdims=True)
    d_out[range(m),Y] -= 1
    #Variable de capa oculta 2
    d_h2 = np.dot(d_out,w.T)
    #Variable de capa oculta 1
    d_h1 = (1-h1**2)*np.dot(d_h2,w2.T)
    
    #Derivada sobre w
    dw = np.dot(h2.T,d_out)
    #Derivada sobre b
    db = d_out.sum(0)
    #Derivada sobre w2
    dw2 = np.dot(h1.T,d_h2)
    #Derivada sobre b2
    db2 = d_h2.sum(0)
    #Derivada sobre w1    
    dw1 = np.dot(X.T,d_h1)
    #Derivada sobre b1
    db1 = d_h1.sum(0)
    #Derivada regularizacin
    dw += reg*w
    dw2 += reg*w2
    dw1 += reg*w1
    
    #Actualizacin de pesos
    w -= l*dw
    b -= l*db
    w2 -= l*dw2
    b2 -= l*db2
    w1 -= l*dw1
    b1 -= l*db1
    
    #Clculo de errores cometidos
    error = ((np.argmax(f, axis=1)-Y)**2).sum(0)
    if t%10 == 0: 
        print('poca', t, '\t Error', error, np.argmax(f, axis=1), '\t Entropy', -np.log(f[range(m),Y]).sum(0))
    
    #Aumenta la iteracin
    t += 1
    #Condicin de paro
    if error == 0 or t == max_it:
        stop = True
        print('poca', t, '\t Error', error, np.argmax(f, axis=1), '\t Entropy', -np.log(f[range(m),Y]).sum(0))
