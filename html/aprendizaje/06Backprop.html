El algoritmo de retropropagacin nos permite entrenar una red neuronal sin importar el nmero de capas que sta red puede tener. Sin este algoritmo, el aprendizaje profundo no sera posible. De all su relevancia.

Creación de la red profunda¶

Este problema puede resolverse con una sola capa oculta. De hehco, los teoremas de aproximador universal nos dicen que una sola capa oculta basta para resolver cualquier problema que una red neuronal puede resolver, siempre y cuando se tenga la anchura suficiente en la red.

En la prctica, es comn distribuir el cmputo en diversas capas, en lugar de tener una sola capa con una anchura muy grande. En este ejemplo simple, podemos usar una red profunda para solucionar el problema de clasificacin. Si bien el ejemplo no requiere de muchas neuronas, podemos ver cmo trabajan las redes profundas.

Para simplificar la creacin de la red neuronal profunda, asumiremos que cada capa tiene el mismo nmero de neuronas, toda capa oculta se activar con la funcin tanh, para poder simplificar el clculo de las derivadas.

La red neuronal contar entonces con:

  1. Pesos de la capa de entrada hacia la primera capa oculta, que dependern del tamao de los datos y de la anchura de la red.
  2. $K$ (n_hidfden) capas ocultas y sus correspondientes pesos. En este caso, para simplificar, cada capa ser de matriz de pesos ser de tamao $width \times width$ donde width es la anchura elegida.
  3. Una capa de salida con tantas neuronas de salida como clases.

El esbozo de la clase con el mtodo para computar el forward es el siguiente:

class DeepNN():
    """Clase para entrenar redes neuronales profundas"""
    def __init__(self,n_input,n_output,n_hidden,width):
        super(DeepNN, self).__init__()
        #Pesos capa de entrada
        self.w_in = np.random.rand(n_hidden, n_input)/np.sqrt(n_input)
        self.b_in = np.ones(n_hidden)
        #Pesos para las capas ocultas
        self.w = np.random.rand(width, n_hidden, n_hidden)/np.sqrt(n_input)
        self.b = np.ones((width,n_hidden))
        #Pesos para la capa de salida
        self.w_output = np.random.rand(n_output, n_hidden)/np.sqrt(n_hidden)
        self.b_output = np.ones(n_output)

    def __call__(self, x):
        """Aplicacin de la red neuronal a datos de entrada"""
        # Guarda las pre-activaciones y activaciones en cada capa
        a = np.zeros((layer_width, number_of_hidden_layers))
        h = np.zeros((layer_width, number_of_hidden_layers))
        for k in range(0, layer_width):
            if k == 0:
                # Capa de entrada
                a[k] = np.dot(self.w_in, x) + self.b_in
                h[k] = np.tanh(a[k])
            else:
                # Capas ocultas
                a[k] = np.dot(self.w[k], h[k-1]) + self.b[k]
                h[k] = np.tanh(a[k])

        #Capa de salida
        a_out = np.dot(self.w_output, h[-1]) + self.b_output
        # Funcin softmax
        tmp = np.exp(a_out)
        f = tmp/tmp.sum(0)

        return f

Entrenamiento con backpropagation¶

El entrenamiento es la parte ms compleja. En el entrenamiento debemos optimizar los pesos con base a una funcin de riesgo que, en este caso, ser la entropa cruzada:

$$R = -\sum_i \delta_{y,i} \ln f_i(x)$$

donde $f_i(x)$ es la $i$-sima neurona de salida de la red y $\delta_{y,i}$ se define como:

$$\delta_{y,i} = \begin{cases} 1 & \text{si } y=i \\ 0 & \text{en otro caso} \end{cases}$$

El algoritmo de backpropagation se determina de la siguiente forma:

Donde, para este caso particular, la variable de salida depende de la entropa cruzada y la activacin Softmax, por lo que es:

$$d_{out}(j) = \frac{\partial R(\theta)}{\partial f_j} \frac{\partial f_j}{\partial a^{(out)}} = f_j - \delta_{y,j}$$

Para las capas ocultas, como usamos activacin de tangente hiperblica, las variables de backpropagation son de la forma:

$$d_{k}(j) = \big(1-(h^{(k)})^2\big) \sum_q W^{(k+1)}_{j,q} d_{k+1}(q)$$

Se debe crear un mtodo train que calcule estas variables y actualice los pesos por cada capa. el esbozo de este mtodo se presenta a continuacin:
