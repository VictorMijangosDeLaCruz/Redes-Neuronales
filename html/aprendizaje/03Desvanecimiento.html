Desvanecimiento del gradiente¶

The Vanishing Gradient Problem

Problema

Para funciones de activacin, cuya derivada se acerca a cero, conforme se agregan capas a la red neuronal este se vuelve ms y ms difcil de entrenar.

Los algoritmos de optimizacin basados en el gradiente comparten la idea escencial del descenso por el gradiente: \begin{align} W_{t+1} = W_t - \alpha \nabla_W J \end{align} donde $J$ es la funcin de error y $W$ los pesos.

Recordando que para calcular el gradiente con respecto a cada capa de pesos ste es proporcional a: \begin{align} \Delta^{(l-1)} &= \Delta^{(l)}\left(W^{(l-1)}\right)^T \circ g'(Z^{(l-1)}) \end{align}

tenemos que, entre ms capas, ms veces aparece el trmino $g'(Z^{(l-1)})$, de modo que para la primer capa:

\begin{align} \Delta^{(1)} &\propto g'(Z^{(0)}) \circ g'(Z^{(1)}) \circ ... \circ g'(Z^{(L-1)}) \end{align}

siendo todos estos nmeros $\ll 1$.

De este modo, al sustituir el valor de un gradiente muy pequeo en la frmula de entrenamiento, los pesos prcticamente no cambian, independientemente de qu tan grande sea la magnitud del valor de entrada.

class DeepNeuralNetwork(nn.Module):
    def __init__(self, s, f_activacion=nn.Sigmoid):
        """
        param s: Lista de nmero de neuronas en las capa ocultas
        """
        super(DeepNeuralNetwork, self).__init__()
        
        # Primer capa oculta:
        componentes = [nn.Linear(1,s[0]), f_activacion()]
        
        # Dems capas ocultas:
        for i in range(1,len(s)):
            componentes.append(nn.Linear(s[i-1],s[i]))
            componentes.append(f_activacion())
        
        # Capa de salida
        componentes.append(nn.Linear(s[-1],1))
        componentes.append(f_activacion())
        
        # Arma perceptrn multicapa
        self.layers_stack = nn.Sequential(*componentes)
        
    def forward(self, x):
        """
        Evaluacin de la red sobre la entrada x
        usando alimentacin hacia adelante para los pesos
        actuales.
        """
        logits = self.layers_stack(x)
        return logits

def linealiza_params(params):
    params_list = []
    grad_list = []
    for t in params:
        nparams = torch.reshape(t, (-1,)).detach().numpy()
        for p in nparams:
           params_list.append(p)
        ngrads = torch.reshape(t.grad, (-1,)).detach().numpy()
        for g in ngrads:
            grad_list.append(g)
    return params_list, grad_list

def train(X, Y, model, learning_rate, num_steps):
    """
    Funcin que realiza el entrenamiento:
    Intentar reducir la distancia entre los valores que salen
    de la red y lo que deseamos modificando los pesos de la red.
    """
    size = len(X)
    loss_fn = nn.MSELoss()
    # Misma alfa para todos los pesos:
    #optimizer = torch.optim.SGD(model.parameters(), lr=learning_rate)
    
    # Utiliza momentos para actualizar alfa
    # sobre parmetros individuales
    optimizer = torch.optim.Adam(model.parameters(), lr=learning_rate)
    step_report = 100
    num_reports = int(math.ceil(num_steps/step_report))
    e = np.zeros(num_reports)
    for i in range(num_steps):
        # Prediccin y error
        Y_hat = model(X)
        loss = loss_fn(Y_hat, Y)
        
        # Reduciendo el error
        optimizer.zero_grad()
        loss.backward()
        optimizer.step()
        
        # Cmo vamos
        if i % step_report == 0:
            num_report = int(i/step_report)
            e[num_report] = loss.item()
            #print(f"{num_report} loss: {loss:>7f}")
    #plt.plot(e)
    return Y_hat, e
