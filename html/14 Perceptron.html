Interpretación de pesos y estimación logística¶

En las redes neuronales, los pesos pueden interpretarse como la fuerza de las conexiones que unen dos neuronas entre s. De esta forma, los pesos de la red neurona determinan cmo se comportar sta cuando reciba un vector de entrada: de los pesos depende la clase a la que pertenecer el ejemplo de entrada.

Sin duda, cuanto ms compleja sea la red, ms compleja ser la interpretacin de las interacciones de las neuronas a travs de los pesos. Aqu presentamos dos ejemplos simples: un perceptrn y una red FeedForfard con una sola capa. A partir de esto, podemos entender qu funcin desempean los pesos en las redes neuronales.

Preparación de los datos¶

Una parte immportante de toda red neuronal son los datos con los que va a entrenarse. Estos datos le ayudarn a ajustar sus parmetros (pesos) y, de esta forma, crear una funcin capaz de generalizar la tarea que ha aprendido.

Los datos pueden verse como vectores aleatorios; es decir, como vectores cuyas entradas son variables aleatorias. As, un vector aleatorio se ve como:

$$X^T = [X_1, X_2, ..., X_n]$$

donde $X_i$, para toda $i=1,...,n$, es una variable aleatoria. Cuando muestreamos los ejemplos, esperamos que estos sean iid (idependientes e idnticamente distribuidos); esto implica que cada vector ejemplo $X$ no depende de ningn otro ejemplo en el dataset. Nuestro objetivo, entonces es determinar el valor que una variable $Y$ puede tomar dado que conocemos el vector $X$. Este problema se puede plantear como:

$$\hat{y} = \arg\max_y p(Y=y|X_1,X_2,...,X_n)$$

En el ejemplo que planteamos a continuacin, elegimos variables que aleatorias $X_1, X_2, ..., X_7$ que responden a preguntas que pueden responderse como s o no. As, $X_1$ es una variable que codifica la pregunta "es animal?": si la respuesta es s, se coloca un 1, si es no se coloca un 0.

Perceptrón¶

El perceptrn es un algoritmo de aprendizaje estadstico que busca estimar las clases de un conjunto de datos a partir de la funcin de decisin:

$$f(x; \theta) = \begin{cases} 1 & \text{ si } wx + b > 0 \\ 0 & \text{ si } wx + b \leq 0\end{cases}$$

Aqu $x$ es un vector de entrada. La funcin f depende de $\theta = {w,b}$ que representa los parmetros o los pesos de la red: $w$ contienen los pesos de las conexiones y $b$ es el bias. Definimos esta funcin a continuacin:

In [9]:
#Funcin de decisin
f = lambda x, theta: (np.dot(x,theta) >= 0).astype(np.int)

Algoritmo del perceptrón¶

El objetivo del algoritmo del perceptrn es obtener los parmetros $\theta$ adecuados para que la red pueda clasificar adecuadamente los ejemplos. Este algoritmo es iterativo y busca minimizar el cunto se equivoca la red en cada paso.

Antes de entrar de lleno al este algoritmo, agregaremos una columna ms a la matriz de datos de entrenamiento; esto con el objetivo de que esta columna represente el bias, de tal forma que slo actualicemos sobre un vector de parmetros:

Para inicializar el algoritmo del perceptron, tomaremos un vector $w \in \mathbb{R}^{n+1}$, donde $n$ es el nmero de variables (la dimensin de los vectores ejemplos) y el $+1$ responde al bias. Asimismo, estableceremos un nmero mximo de iteraciones (en caso de que el algoritmo no sea capaz de minimizar el error) y, finalmente, estableceremos un rango de aprendizaje.

Para entrenar la red, utilizaremos el algoritmo del perceptrn. Este consta de dos partes:

  • Forward. Para cada ejemplo $x$, se computar la funcin $f(x; \theta)$
  • Backward. Se actualizarn los pesos de $\theta = {w,b}$ por medio de la regla: $$w_i \leftarrow w_i - \eta (f(x;\theta)-y_x)x_i$$ Donde $y_x$ es la clase a la que pertenence el ejemplo $x$.

El algoritmo iterar hasta que el error $f(x; \theta) -y$ sea 0 o cuando se cumpla el nmero mximo de iteraciones.

  • Converge a 1 cuando $x\to\infty$
  • Converge a 0 cuando $x\to -\infty$

En este sentido es una funcin con valores en $[0,1]$. Tomaremos al sigmoide como una funcin de probabilidad. En este sentido, la estimacin de probabilidad a aprtir del perceptrn puede verse como:

$$p(Y=1|x) = \sigma(wx+b)$$

Por su parte, la probabilidad de la clase negatica est dada por:

$$p(Y=0|x) = 1-\sigma(wx+b) = \sigma(-wx-b)$$

Esto euivale a proyectar los puntos obtenidos con la preactivacin del perceptrn $wx+b$ en la funcin sigmoide, como puede verse a continuacin: as, los puntos ms alejados de 0 positivamente quedarn en la seccin del sigmoide ms cercana a 1, mientras que los que se alejan negativamente quedarn en la seccin del sigmoide ms cercana a 0.

Problema de estimación en regresión logística¶

Cuando estimamos una regresin logstica en base a un conjunto de datos $X$ y sus clases $Y$, lo que se busca es encontrar los parmetros $\theta = \{w, b\}$ que mejor aproximen el comportamiento de los datos.

En este caso, se tiene que minimizar una funcin de entropa dada por la funcin de riesgo:

$$R(\theta) = -\sum_x [y_x \log \sigma(wx+b) + (1-y_x) \log \sigma(wx+b)]$$

En este caso, $y_x$ representa la clase verdadera del ejmplo $x$. As, esta funcin de riesgo puede encontrar su mnimo en la solucin de los parmetros ptima, lo que dara las probabilidades estimadas que mejor describan la distribucin de los datos.

Podemos observar el comportamiento de la funcin de riesgo sobre los datos (tomamos slo las dimensiones necesarias para visualizacin).

    H = (-y_real*np.log(p_hat) - (1-y_real)*np.log(p_hat)).sum() 
    return H
