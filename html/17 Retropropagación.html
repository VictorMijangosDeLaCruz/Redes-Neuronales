Nuestra red se compondr√° de:

  • 3 capas ocultas, cada una con 10 unidades ocultas y activaci√≥n $\tanh$.
  • Una capa de salida con tres neuronas (clases de los datos) con activaci√≥n Softmax.

Entrenamiento con backprop y SGD¶

El entrenamiento consta de dos pasos:

  • Forward: Se calcula el valor de la salidad pasando por las capas ocultas:
    • Preactivaci√≥n: $a^{(k)} = W^{(k)}h^{(k-1)}+b^{(k)}$, con $h^{(0)} = x$
    • Activaci√≥n: $h^{(k)} = \tanh(a^{(1)})$
    • Preactivaci√≥n en salida: $a^{(out)} = W^{(out)}h^{(3)}+b^{(out)}$
    • Activaci√≥n en salida: $f = Softmax\big(a^{(out)}\big)$

  • Bacward: El backward utiliza el algoritmo de backpropagation y el SGD para actualizar. Se calculan las variables de backprop, en base a la funci√≥n de riesgo $$R(\theta) = -\sum_x \sum_y y \ln f(x)$$ Las variables son:

    • $d_{out}(j) = \frac{\partial R(\theta)}{\partial f_j} \frac{\partial f_j}{\partial a^{(out)}} = f_j - \delta_{y,j}$
    • $d_{k}(j) = \frac{\partial h^{(k)}}{\partial a^{(k)}} \sum_q W^{(k+1)}_{j,q} d_{k+1}(q)$

    En este caso, $\frac{\partial h^{(k)}}{\partial a^{(k)}} = 1-\tanh^2(a^{(k)})$. Y tenemos que: $$\delta_{y,j} = \begin{cases} 1 & \text{si } y=j \\ 0 & \text{en otro caso} \end{cases}$$

#Guarda la p√©rdida
loss = []
#Datos para entrenar
train_data = list(zip(x_train,y_train))

for t in tqdm(range(epochs)):
    #Aleatoriza los datos de entrenamiento
    np.random.shuffle(train_data)
    #Guarda el riesgo por √©poca
    risk_epoch = 0
    for x,y in train_data:
        #FORWARD
        #Primera capa
        a_1 = np.dot(w_1, x) + b_1
        h_1 = np.tanh(a_1)
        #Segunda capa
        a_2 = np.dot(w_2, h_1) + b_2
        h_2 = np.tanh(a_2)
        #Tercera capa
        a_3 = np.dot(w_3, h_2) + b_3
        h_3 = np.tanh(a_3)
        #Capa de salida
        a_out = np.dot(w_output, h_3) + b_output
        tmp = np.exp(a_out)
        f = tmp/tmp.sum(0)
        
        #Riesgo en la √©poca
        risk_epoch -= np.log(f)[int(y)]
        
        #BACKWARD
        #Variables de backprop
        #Variable en la salida
        d_out = f.copy()
        d_out[int(y)] -= 1
        #Variables en las capas ocultas
        d_3 = (1-h_3**2)*np.dot(w_output.T, d_out)
        d_2 = (1-h_2**2)*np.dot(w_3.T, d_3)
        d_1 = (1-h_1**2)*np.dot(w_2.T, d_2)
        
        #Actualizaci√≥n de pesos con SGD
        w_output -= lr*np.outer(d_out, h_3)
        w_3 -= lr*np.outer(d_3, h_2)
        w_2 -= lr*np.outer(d_2, h_1)
        w_1 -= lr*np.outer(d_1, x)
        #Actualizaci√≥n de bias con SGD
        b_output -= lr*d_out
        b_3 -= lr*d_3
        b_2 -= lr*d_2
        b_1 -= lr*d_1
    
    #Guarda la p√©rdida
    loss.append(risk_epoch)

def predict(X):
    """
    Funci√≥n de predicci√≥n de la red neuronal.
    
    Arguments
    ---------
    X : array
        Arreglo de datos de los que se va a predecir
        
    Returns
    -------
        Clases a la que pertenecen los datos seg√∫n la red.
    """
    #Aplica la red
    a_1 = np.dot(X, w_1.T) + b_1
    h_1 = np.tanh(a_1)
    a_2 = np.dot(h_1, w_2.T) + b_2
    h_2 = np.tanh(a_2)
    a_3 = np.dot(h_2, w_3.T) + b_3
    h_3 = np.tanh(a_3)
    a_out = np.dot(h_3, w_output.T) + b_output
    tmp = np.exp(a_out)
    f = tmp/tmp.sum(1, keepdims=True)
    
    #Regresa el argumento que maximiza el Softmax
    return np.argmax(f, axis=1)

def predict_proba(X):
    """
    Funci√≥n de estimaci√≥n de probabilidad de la red neuronal.
    
    Arguments
    ---------
    X : array
        Arreglo de datos de los que se va a predecir
        
    Returns
    -------
        Probabilidad de clase
    """
    #Aplica la red
    a_1 = np.dot(X, w_1.T) + b_1
    h_1 = np.tanh(a_1)
    a_2 = np.dot(h_1, w_2.T) + b_2
    h_2 = np.tanh(a_2)
    a_3 = np.dot(h_2, w_3.T) + b_3
    h_3 = np.tanh(a_3)
    a_out = np.dot(h_3, w_output.T) + b_output
    tmp = np.exp(a_out)
    f = tmp/tmp.sum(1, keepdims=True)
    
    #Regresa la probabilidad por clase
    return f
